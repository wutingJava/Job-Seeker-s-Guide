# 《求职者攻略》游戏策划方案 v1.3

## 一、项目概述

### 1.1 游戏定位
一款以社会宏观经济为模拟核心的策略模拟游戏。玩家扮演社会管理者，通过政策调控、基础建设、应对随机事件来推动人类社会从原始劳动时代演进至大智能时代。

### 1.2 核心体验
- **宏观策略**：俯视整张社会地图，调控经济与就业
- **演进史诗**：见证人类劳动形态的六个时代变迁
- **供需博弈**：平衡人才供给与企业需求的动态关系

### 1.3 技术选型
- **引擎**：JMonkeyEngine 3.x（Java）
- **语言**：Java 17+
- **美术资源**：AI生成 + 几何图形图标
- **目标平台**：PC（Windows/macOS/Linux）

---

## 二、游戏时代系统

### 2.1 六大时代演进

**核心设计理念**：时代跃迁是**全社会的技术升级**，它同时改变所有七大行业的生产方式和人才需求，而非简单地"新增/删除"某些行业。

| 时代 | 名称 | 标志性技术 | 对全行业的影响 | 人才需求特征 |
|------|------|------------|----------------|--------------|
| 1 | 原始劳动时代 | 石器、火、简单工具 | 所有行业依赖人力，体力最重要 | 极低学历，经验主导 |
| 2 | 手工场时代 | 铁器、畜力、文字 | 手工技艺与商业制度形成 | 低学历，学徒制 |
| 3 | 蒸汽革命时代 | 蒸汽机、煤炭、铁路 | 机械化替代人力，工厂出现 | 中等技能，机械操作 |
| 4 | 电气时代 | 电力、内燃机、化工 | 大规模生产与电气化管理 | 中高学历，工程师阶层 |
| 5 | 信息时代 | 计算机、互联网、半导体 | 自动化+信息化，知识经济 | 高学历，信息技术 |
| 6 | 大智能时代 | AI、机器人、量子计算 | AI替代重复劳动，创造力为王 | 极高学历，科研+AI |

**每个时代的关键跃迁挑战**：
- 每个行业的岗位要求同时提升（学历、技能门槛提高）
- 部分传统岗位消失（如马夫），但同行业的新岗位出现（如司机）
- 玩家必须在短时间内完成人才结构调整
- 失败代价：大量低技能劳动者失业，引发社会动荡

### 2.2 时代演进条件

每个时代向下一时代演进需满足：
- **科技进步值**达到阈值（如信息时代需科技进步≥80%）
- **产业结构合理**：旧时代岗位占比下降到合理水平
- **就业率**保持稳定（≥70%）
- **触发事件**：标志性技术突破事件（如"互联网发明"、"ChatGPT诞生"）

## 三、经济系统

### 3.1 核心经济指标

| 指标 | 说明 | 正常范围 |
|------|------|----------|
| GDP | 国内生产总值 | 0-∞ |
| 通胀率 | 物价上涨速度 | -5% ~ 20% |
| 失业率 | 求职失败/总劳动力 | 0% ~ 50% |
| 基尼系数 | 社会公平度 | 0.1 ~ 0.9 |
| 科技进步率 | 技术发展速度 | 0% ~ 100% |
| 幸福指数 | 社会满意度 | 0 ~ 100 |

### 3.2 供需模型

**人才供给侧**：
- 新增劳动力（每年自然增长）
- 院校培养速度
- 培训补贴影响

**企业需求侧**：
- 企业数量（受投资环境影响）
- 行业景气指数（周期性波动）
- 薪资期望（随时代升高）

**匹配算法**：
```
就业成功率 = f(学历匹配度, 技能匹配度, 薪资合理性, 运气值)
```

### 3.3 详细数值模型

**企业招聘满意度公式**：
```
企业招聘满意度 = 0.3×学历匹配度 + 0.4×技能匹配度 + 0.2×薪资合理性 + 0.1×经验匹配度
```

**求职者就业概率公式**：
```
就业概率 = 基础匹配度 × 运气系数 × (1 + 就业促进buff) × 行业景气系数
```

**科技进步公式**：
```
科技进步 = Σ(研发投入 × 研发效率系数) + 随机事件修正 + 人才溢出效应
```

**GDP增长公式**：
```
GDP增长 = Σ(就业人口 × 人均产出) × (1 + 科技进步加成) - 失业人口消耗 - 通胀损耗
```

## 四、人才系统

### 4.1 求职者属性

每个求职者拥有四个维度属性（1-100）：

| 属性 | 说明 | 对就业影响 |
|------|------|------------|
| 学历 | 教育程度 | 影响高薪岗位解锁 |
| 技能 | 专业技能水平 | 影响岗位匹配度 |
| 经验 | 工作年限 | 影响薪资期望 |
| 运气 | 随机因素 | 不可控波动 |

### 4.2 求职者生命周期
1. **出生** → 进入待就业池
2. **求学阶段** → 可选择投入教育资源
3. **求职阶段** → AI自动匹配或受政策buff影响
4. **就业/失业** → 就业者产出GDP，失业者消耗社会资源
5. **退休/死亡** → 退出劳动力市场

### 4.3 院校与培训
- **小学/中学**：提升基础学历
- **职业学院**：提升特定技能
- **大学**：高学历 + 宽泛技能
- **研究所**：科研人才产出

### 4.4 人才流动机制
- **人才引进**：玩家可花费资源从"国际市场"引进人才（加速但昂贵）
- **人才流失**：政策不当会导致本地人才流向其他地区
- **海归潮**：特定事件触发海外人才回归

**设计理念**：所有行业从原始时代到大智能时代都持续存在，不同的是**技术水平、人才需求、资源消耗**和**生产方式**。行业不会消失，而是不断进化。

| 行业编号 | 行业名称 | 核心功能 |
|----------|----------|----------|
| A | 农业 | 食物与原材料生产 |
| B | 制造业 | 商品加工与制造 |
| C | 商业贸易 | 商品流通与交换 |

### 5.9 行业景气与竞争机制

**每个行业有独立的景气指数**（周期性波动）：
- 繁荣期：企业扩张、薪资上涨、就业需求增加
- 平稳期：稳定经营、薪资持平
- 衰退期：企业收缩、薪资下降、裁员

**企业竞争机制**：
- 同行业企业竞争人才，薪资高的企业更容易招到高技能人才
- 技术先进的企业生产力更高、利润更高
- 玩家政策选择会影响不同行业的兴衰

**企业生命周期**：
- **初创** → **成长** → **成熟** → **转型/衰退** → **倒闭/被收购**
- 大企业倒闭会引发上下游行业的连锁反应

**行业结构转型的关键游戏机制**：
- 当农业人口从90%降到2%时，这2%的农业从业者必须具备更高技能和学历
- 释放出来的劳动力必须被其他行业（制造业、服务业、科技业）吸收
- 玩家的挑战是：在时代跃迁期间，确保教育系统培养出足够多的高技能人才，同时通过政策引导企业转型，避免大规模失业和社会动荡

## 六、政府财政系统

### 6.1 财政收支基本框架

**核心理念**：政府如同一个超级企业，有完整的收入来源和支出项目，必须保持收支平衡。财政赤字会积累债务，过高的债务会引发债务危机。

**财政周期**：每游戏年度结算一次（12个月）

---

### 6.2 政府收入来源

| 收入类别 | 子项 | 说明 | 可调参数 |
|----------|------|------|----------|
| **个人所得税** | 工资税 | 按工资收入的百分比征收，可设为累进税率 | 基础税率(0-50%)、起征点、累进档位 |
| | 资本利得税 | 对投资收益征税 | 税率(0-40%) |
| **企业所得税** | 企业利润税 | 按企业利润百分比征收 | 税率(0-40%) |
| | 行业差别税 | 对特定行业加征或减免 | 行业税率加成 |
| **消费税/增值税** | 商品服务税 | 在商品/服务消费环节征收 | 基础税率(0-25%)、民生商品减免 |
| | 奢侈品税 | 对高消费商品额外征税 | 高消费税率(0-50%) |
| **公共服务收费** | 教育收费 | 公立学校学费（小学→大学各有定价） | 各阶段学费 |
| | 交通收费 | 公交、地铁、铁路票价 | 各线路票价、区间定价 |
| | 医疗收费 | 公立医院挂号费、诊疗费、药费 | 基础诊疗费、药品加成 |
| | 水电收费 | 居民/工业水电费 | 居民电价、工业电价、阶梯定价 |
| | 其他服务 | 公园、图书馆、体育设施等 | 各服务定价 |
| **资源收入** | 土地出让金 | 企业用地拍卖收入 | 土地出让政策 |
| | 资源开采费 | 矿产/能源资源开采许可 | 资源税率 |
| **债务收入** | 国债发行 | 发行债券融资（非经常性收入） | 发债额度、利率 |

**实时计算公式**：
```
政府月收入 = Σ(个人所得税) + Σ(企业所得税) + Σ(增值税)
           + Σ(公共服务收费) + Σ(资源收入) + 国债发行
```

---

### 6.3 政府支出项目

| 支出类别 | 子项 | 说明 | 影响因素 |
|----------|------|------|----------|
| **教育支出** | 公立学校运营 | 校舍维护、设备、教师工资 | 学生数量×生均成本 |
| | 教师薪酬 | 教师和管理人员工资 | 教师人数×教师月薪 |
| | 教育补贴 | 对困难学生的助学金、奖学金 | 补贴覆盖率×人均补贴 |
| | 教育基建 | 新建学校、改造校舍 | 建设数量×建造成本 |
| **交通支出** | 公交系统运营 | 公交司机工资、车辆维护、燃料 | 运营线路数×线路成本 |
| | 道路维护 | 公路、桥梁维护费用 | 道路里程×单位维护成本 |
| | 交通基建 | 新建铁路、地铁、公路 | 建设项目×项目成本 |
| **医疗支出** | 公立医院运营 | 医生工资、药品采购、设备维护 | 就诊人数×人均成本 |
| | 医疗补贴 | 医保报销、贫困人口免费医疗 | 报销比例×医疗总支出 |
| | 公共卫生 | 疫苗接种、疾病防控、健康教育 | 人口数×人均卫生费用 |
| **社会保障支出** | 养老金 | 退休人员养老金 | 退休人数×人均养老金 |
| | 失业保险金 | 失业者每月救济金 | 失业人数×失业救济金 |
| | 低保/福利 | 最低生活保障 | 贫困人口×低保金额 |
| **基建支出** | 公共设施建设 | 公园、图书馆、文化设施 | 建设数量×建设成本 |
| | 基础设施投资 | 电网、水利、通信等 | 投资项目数×项目成本 |
| **科研支出** | 科研经费 | 公立研究机构运营费用 | 科研人员数×人均经费 |
| | 科研项目补贴 | 对特定科技领域的研究补贴 | 补贴金额 |
| **公共安全支出** | 警察/消防 | 警务人员工资、装备、消防 | 警力规模×人均成本 |
| **行政支出** | 政府运营 | 公务员薪酬、办公开支 | 公务员人数×人均行政成本 |
| **债务利息** | 国债利息 | 已有国债的利息支出 | 国债总额×利率 |

**实时计算公式**：
```
政府月支出 = Σ(教育支出) + Σ(交通支出) + Σ(医疗支出)
           + Σ(社会保障) + Σ(基建支出) + Σ(科研支出)
           + Σ(公共安全) + Σ(行政支出) + 债务利息
```

### 6.4 财政平衡与债务系统

**核心公式**：
```
月度财政盈余 = 月度收入 - 月度支出
年度财政赤字 = Σ(月度盈余)  (负值即为赤字)
国债总额 = 上期国债 + 年度赤字 + 新发国债
债务利息 = 国债总额 × 国债利率
```

**债务等级与风险**：

| 债务/GDP比率 | 债务等级 | 影响 |
|---------------|----------|------|
| <30% | 健康 | 融资容易，利率低 |
| 30%-60% | 正常 | 正常融资，利率适中 |
| 60%-100% | 警戒 | 投资者警惕，利率上升 |
| 100%-150% | 高风险 | 难以发新债，利率飙升 |
| >150% | 债务危机 | 被迫财政紧缩，可能触发经济危机事件 |

**国债利率机制**：
- 基础利率由时代和经济状况决定
- 债务/GDP比率越高，新发国债利率越高
- 高利率时代（如电气时代）债务负担更重
- 低利率时代（如信息时代早期）发债成本低

### 6.5 公共服务定价与需求反馈机制

**核心设计**：每个公共服务有一个玩家可调整的价格，价格高低直接影响市民的使用意愿和行为决策。

#### 6.5.1 教育定价模型

```
学生上学意愿 = 基础意愿
             + 教育质量加成(受教育投入影响)
             - 学费压力(学费/家庭收入 × 价格敏感度)
             + 家长期望加成
             + 政策激励(如义务教育免费)
```

**各阶段教育特征**：

| 教育阶段 | 参考月学费范围 | 需求价格弹性 | 入学率影响因素 | 时代解锁 |
|----------|----------------|--------------|----------------|----------|
| 小学 | 免费 ~ 家庭收入5% | 低（刚需） | 义务教育政策、学校覆盖率 | 原始劳动时代 |
| 中学 | 免费 ~ 家庭收入10% | 中低 | 学费、教育质量 | 手工场时代 |
| 职业学院 | 家庭收入5% ~ 20% | 中 | 就业前景、学费补贴 | 蒸汽革命时代 |
| 大学 | 家庭收入15% ~ 40% | 高 | 学费、奖学金、就业前景 | 电气时代 |
| 研究所/博士 | 家庭收入30% ~ 60% | 极高 | 全额奖学金、科研前景 | 信息时代 |

**玩家策略空间**：
- **免费义务教育**：提高入学率，但财政负担重
- **收费教育**：减轻财政压力，但低收入家庭学生可能辍学
- **奖学金制度**：部分补贴贫困学生，兼顾公平与财政

**政策效果链**：
```
提高学费 → 政府收入↑ / 财政负担↓
       → 低收入家庭学生入学率↓
       → 长期人才供给↓
       → 产业升级受阻
       → 幸福指数↓
```

```
降低学费/免费 → 政府收入↓ / 财政负担↑
              → 入学率↑
              → 人才培养速度↑
              → 幸福指数↑
              → 但财政可能陷入赤字
```

#### 6.5.2 公共交通定价模型

```
市民出行意愿 = 基础出行需求
             + 交通便利度(线路覆盖率、速度)
             - 票价压力(票价/家庭收入 × 价格敏感度)
             - 拥挤程度(高峰期拥挤度)
             + 替代交通成本(私家车成本、打车成本)
```

**交通定价策略**：

| 策略 | 票价水平 | 效果 | 财政影响 |
|------|----------|------|----------|
| 补贴型公交 | 低于成本 | 鼓励出行、减少拥堵、环保 | 政府需补贴差额 |
| 成本定价 | 约等于运营成本 | 收支平衡、出行中等 | 补贴少量或不补贴 |
| 盈利型公交 | 高于成本 | 政府盈利、但低收入者出行受限 | 产生利润 |
| 免费公交 | 0 | 出行意愿暴涨、但财政压力极大 | 全由政府承担 |

**政策效果链**：
```
提高公交票价 → 政府收入↑
           → 出行量↓
           → 企业员工通勤成本↑(间接影响企业成本)
           → 私家车购买↑(信息时代后出现)
           → 拥堵和环境污染↑
           → 幸福指数↓
```

#### 6.5.3 医疗定价模型

```
就医意愿 = 健康需求
         - 医疗费用压力(诊疗费/家庭收入 × 健康敏感度)
         + 医保报销比例
         - 就医便利度(医院距离、排队时间)
```

**医疗定价策略**：

| 策略 | 诊疗费水平 | 医保报销 | 效果 | 财政影响 |
|------|------------|----------|------|----------|
| 全民免费医疗 | 极低或0 | 100% | 就医率高、健康水平高 | 财政负担极重 |
| 医保制度 | 中等 | 50-80% | 平衡财政与健康 | 中等负担 |
| 自费为主 | 较高 | 低或无 | 就医率低、贫富健康差距 | 负担轻 |

**政策效果链**：
```
提高诊疗费 → 政府负担↓
          → 就医率↓
          → 人口健康水平↓
          → 劳动力质量下降、死亡率上升
          → 贫富健康差距↑
          → 幸福指数↓
```

#### 6.5.4 水电定价模型

```
水电使用量 = 基础生活需求
           + 企业生产需求
           - 价格压力(电价/家庭收入 × 能源敏感度)
           + 季节因素(冬季采暖、夏季制冷)
```

**阶梯定价策略**：
- 第1档（基础用量）：低价，保障民生
- 第2档（正常用量）：中等价格
- 第3档（超额用量）：高价，节约能源

**工业/居民差别定价**：
- 居民用电：稳定、低价（保障民生）
- 工业用电：根据产业政策调整（高耗能行业可提高电价）

### 6.6 税收系统详解

#### 6.6.1 个人所得税（累进税制）

```
应税收入 = 工资收入 - 免税额
应缴税款 = 按累进税率表计算
```

**累进税率表示例（可由玩家调整）**：

| 收入档位 | 税率 | 时代适用性 |
|----------|------|------------|
| 低于最低工资 × 2 | 0% | 全时代 |
| 最低工资×2 ~ ×5 | 5-15% | 全时代 |
| 最低工资×5 ~ ×10 | 15-30% | 手工场+ |
| 最低工资×10 ~ ×20 | 30-45% | 蒸汽+ |
| 超过最低工资×20 | 45-60% | 电气+ |

**玩家可调参数**：
- 基础起征点
- 各档位税率
- 档位划分

**政策效果链**：
```
提高个人所得税 → 政府收入↑
               → 个人可支配收入↓
               → 消费能力↓
               → 企业销售收入↓
               → 幸福指数↓
               → 基尼系数↓(高收入者征税更多)
```

#### 6.6.2 企业所得税

```
企业利润 = 企业收入 - 企业成本(工资、原材料、折旧)
企业应缴税款 = 企业利润 × 企业所得税率
```

**行业差别税率**：
- 基础税率（默认）
- 高耗能行业加成（+5-15%）
- 高新技术行业减免（-5-20%）
- 农业扶持减免（-10-30%）

**政策效果链**：
```
提高企业所得税 → 政府收入↑
               → 企业净利润↓
               → 企业扩张意愿↓
               → 就业机会↓
               → 企业向低税区迁移意愿↑
               → 长期GDP增长↓
```

#### 6.6.3 增值税/消费税

```
商品最终价格 = 生产成本 + 利润 + 增值税
增值税 = (售价 - 进项成本) × 增值税率
```

**玩家可调参数**：
- 基础增值税率
- 民生商品（食品、药品）减免税率
- 奢侈品加征税率

**政策效果链**：
```
提高增值税 → 物价↑
          → 消费能力↓(实际购买力下降)
          → 政府收入↑
          → 企业销量↓
          → 幸福指数↓
```

### 6.8 财政危机与风险事件

#### 6.8.1 自动触发事件

| 触发条件 | 事件 | 玩家应对 |
|----------|------|----------|
| 连续3年财政赤字且债务/GDP > 60% | **财政紧张** | 必须削减支出或加税 |
| 债务/GDP > 100% | **债务危机警告** | 评级下调，新发债券利率↑50% |
| 债务/GDP > 150% | **主权债务危机** | GDP骤降10-20%，必须紧急财政紧缩 |
| 税收过高（总税负>40%GDP） | **抗税运动** | 幸福指数骤降，可能引发社会动荡 |
| 公共服务定价过高 | **抗议活动** | 特定服务使用量骤降，幸福指数下降 |

#### 6.8.2 玩家应对选项

当财政危机发生时，玩家有以下选择：

| 选项 | 效果 | 副作用 |
|------|------|--------|
| **削减教育支出** | 节省资金20-50% | 人才培养速度下降，长期负面影响大 |
| **削减医疗支出** | 节省资金10-30% | 人口健康下降，死亡率上升 |
| **削减养老金** | 节省资金15-40% | 老年人贫困率上升，幸福指数骤降 |
| **提高个人所得税** | 增加收入10-30% | 消费下降，幸福指数下降 |
| **提高企业所得税** | 增加收入10-30% | 企业扩张下降，就业减少 |
| **提高公共服务收费** | 增加收入10-20% | 民生负担增加，使用量下降 |
| **发行新国债** | 获得大额现金 | 债务总额增加，未来利息负担加重 |
| **财政紧缩(综合)** | 强制削减20%所有支出 | 全面社会满意度下降，短期GDP受挫 |

### 6.9 贴近现实的财政反馈循环

#### 6.9.1 教育-人才-产业循环

```
政府教育投入 → 学校质量↑ → 学生技能/学历↑
           → 毕业生能力↑ → 企业生产力↑ → GDP↑ → 税收↑
           → 高学历人才供给↑ → 新兴产业发展↑ → 税收↑
```

**但如果定价过高**：
```
学费↑ → 低收入家庭入学率↓ → 人才断层↑
     → 贫富技能差距↑ → 基尼系数↑ → 社会不稳定↑
```

#### 6.9.2 交通-经济-就业循环

```
政府交通投入 → 交通便利度↑ → 就业可达范围↑ → 就业匹配率↑
           → 企业物流成本↓ → 企业利润↑ → 税收↑
           → 商业活动↑ → 商业贸易行业↑ → GDP↑

但如果票价过高：
票价↑ → 出行量↓ → 就业可达范围↓ → 失业率↑
     → 企业通勤成本↑ → 企业利润↓ → GDP↓
```

#### 6.9.3 医疗-健康-劳动力循环

```
政府医疗投入 → 人口健康水平↑ → 劳动力质量↑ → 生产力↑
           → 平均寿命↑ → 劳动力供给时间↑ → GDP↑

但如果诊疗费过高：
诊疗费↑ → 就医率↓ → 人口健康↓ → 劳动力质量↓ → GDP↓
       → 贫困人口健康状况恶化 → 贫富差距↑
```

#### 6.9.4 税收-增长-公平平衡循环

```
               ┌→ 政府收入↑ → 公共支出↑ → 基础设施↑ ─┐
               │                                        │
税率调整 → 影响 ┼→ 个人可支配收入↓ → 消费↓ → 企业销售↓ →┼→ GDP变化
               │                                        │
               └→ 企业利润↓ → 投资↓ → 就业↓ → 消费↓ ───┘

同时：
高累进税率 → 基尼系数↓ → 社会公平↑ → 幸福指数↑
低税率 → 企业活力↑ → 经济增长↑ → 但可能贫富差距↑
```

### 7.2 综合政策工具

**财政政策**：
- 个人所得税调整（累进税率表）
- 企业所得税调整（含行业差别税）
- 增值税调整（含民生商品减免）
- 公共服务定价（教育/交通/医疗/水电）
- 国债发行与偿还
- 财政赤字上限设定

**教育政策**：
- 各阶段学校学费定价
- 教师薪酬水平
- 教育经费投入占比
- 助学金/奖学金制度
- 义务教育政策（免费但财政负担）

**交通政策**：
- 公交/地铁票价
- 道路建设预算
- 公共交通补贴比例
- 交通拥挤费（信息时代后解锁）

**医疗政策**：
- 公立医院诊疗费
- 医保报销比例
- 药品价格管制
- 公共卫生预算

**社会保障政策**：
- 养老金标准
- 失业救济金
- 最低生活保障
- 退休年龄

**科技政策**：
- 研发补贴
- 技术引进
- 知识产权保护

### 7.3 基础建设

玩家可在地图上建造：

| 建筑 | 建设成本 | 年度维护成本 | 效果 |
|------|----------|-------------|------|
| 学校 | 高 | 中 | 增加人才培养容量 |
| 医院 | 高 | 中高 | 提高医疗服务能力 |
| 公交站/地铁站 | 中 | 中 | 提高交通便利度 |
| 培训中心 | 中高 | 中 | 提升成人技能再培训 |
| 企业园区 | 中 | 低 | 吸引企业入驻 |
| 研发中心 | 高 | 中高 | 加速科技进步 |
| 就业中心 | 中 | 低 | 提升就业匹配率 |
| 图书馆/文化中心 | 中低 | 低 | 提升幸福指数、辅助教育 |

### 7.4 干预限制与政策延迟

| 政策类型 | 延迟时间 | 副作用 |
|----------|----------|--------|
| 税率调整 | 3个月 | 影响消费信心、企业投资 |
| 教育经费投入 | 3-5年 | 人才供给滞后，财政负担 |
| 最低工资 | 1年 | 可能推高失业率 |
| 研发补贴 | 即时+长期 | 财政负担累积 |
| 基建投资 | 2-3年 | 短期无效果、长期收益大 |
| 公共服务调价 | 1个月 | 立即影响使用意愿和幸福指数 |
| 养老金调整 | 6个月 | 老年人满意度变化 |
| 国债发行 | 即时 | 增加未来债务负担 |

### 7.5 区域地理系统

- **地图分区**：将社会地图划分为4-6个区域（如：东区、西区、南区、北区、中心区）
- **区域差异**：不同区域有初始禀赋（资源、人口密度、交通便利度）
- **区域发展不平衡**：玩家可通过基建投资平衡区域发展
- **区域差别政策**：可对不同区域实施不同的交通/教育投资策略
- **区域特色企业**：特定区域有自然产业优势（如港口区→商业贸易，山区→资源型制造业）

## 八、事件系统

### 8.1 随机事件类型

| 类型 | 事件示例 | 影响 |
|------|----------|------|
| 经济危机 | 1929大萧条 | 企业倒闭潮，失业率飙升 |
| 技术突破 | 互联网发明 | 新行业崛起，人才需求剧变 |
| 人口事件 | 婴儿潮/老龄化 | 劳动力供给大幅变化 |
| 社会事件 | 罢工运动 | 生产停滞，经济受损 |
| 政策变革 | 改革开放 | 市场经济激活 |
| **财政事件** | 主权信用评级下调 | 发债成本上升，债务利息增加 |
| **财政事件** | 国际援助 | 获得一次性资金注入 |
| **财政事件** | 金融危机 | 企业大量倒闭，税收骤减 |

### 8.2 事件触发机制

- 随时间推移自动触发
- 特定条件满足时触发（如失业率>30%触发福利改革运动）
- 玩家政策选择触发（如提高失业率触发社会动荡）
- 财政指标触发（如债务/GDP>100%触发债务危机）
- 税收过高触发抗税运动
- 公共服务定价过高触发抗议活动

## 十二、新手引导与难度系统

### 12.1 难度等级

| 难度 | 描述 | 特征 |
|------|------|------|
| 和谐模式 | 新手友好 | 事件温和、预算宽松、失业率容忍度高、财政宽松 |
| 普通模式 | 标准体验 | 正常事件、预算平衡、合理税收压力 |
| 挑战模式 | 硬核模拟 | 随机危机频繁、预算紧张、玩家干预延迟更长、债务利息更高 |

### 12.2 新手引导流程

```
引导流程：
1. 第一年：只显示地图，提示"这是你的社会"
2. 第二年：弹出"需要建设学校培养人才（会产生财政支出）"
3. 第三年：弹出"企业需要工人，查看就业匹配"
4. 第四年：弹出"财政仪表盘——你的收入来自税收，支出用于公共服务"
5. 第五年：弹出"你可以调整税率、学费、公交票价来平衡财政"
6. 第六年：引导结束，正式进入自由模式
```
