# 《求职者攻略》游戏策划方案 v1.1

## 一、项目概述

### 1.1 游戏定位
一款以社会宏观经济为模拟核心的策略模拟游戏。玩家扮演社会管理者，通过政策调控、基础建设、应对随机事件来推动人类社会从原始劳动时代演进至大智能时代。

### 1.2 核心体验
- **宏观策略**：俯视整张社会地图，调控经济与就业
- **演进史诗**：见证人类劳动形态的六个时代变迁
- **供需博弈**：平衡人才供给与企业需求的动态关系

### 1.3 技术选型
- **引擎**：JMonkeyEngine 3.x（Java）
- **语言**：Java 17+
- **美术资源**：AI生成 + 几何图形图标
- **目标平台**：PC（Windows/macOS/Linux）

## 二、游戏时代系统

### 2.1 六大时代演进

| 时代 | 名称 | 核心特征 | 典型岗位 | 技术标志 |
|------|------|----------|----------|----------|
| 1 | 原始劳动时代 | 体力劳动为主 | 农民、猎人、匠人 | 简单工具 |
| 2 | 手工场时代 | 技艺传承 | 工匠、商人、学徒 | 手工业 |
| 3 | 蒸汽革命时代 | 机械化生产 | 工人、工程师 | 蒸汽机 |
| 4 | 电气时代 | 大规模生产 | 技术员、管理者 | 电力、内燃机 |
| 5 | 信息时代 | 知识经济 | 程序员、分析师 | 计算机、互联网 |
| 6 | 大智能时代 | AI协作 | 科学家、创新者 | 人工智能、机器人 |

### 2.2 时代演进条件
每个时代向下一时代演进需满足：
- **科技进步值**达到阈值（如：信息时代需科技进步≥80%）
- **GDP人均值**达到阈值
- **就业率**维持稳定（≥70%）
- **触发事件**：标志性历史事件（如"互联网发明"、"ChatGPT诞生"）

## 三、经济系统

### 3.1 核心经济指标

| 指标 | 说明 | 正常范围 |
|------|------|----------|
| GDP | 国内生产总值 | 0-∞ |
| 通胀率 | 物价上涨速度 | -5% ~ 20% |
| 失业率 | 求职失败/总劳动力 | 0% ~ 50% |
| 基尼系数 | 社会公平度 | 0.1 ~ 0.9 |
| 科技进步率 | 技术发展速度 | 0% ~ 100% |
| 幸福指数 | 社会满意度 | 0 ~ 100 |

### 3.2 供需模型

**人才供给侧**：
- 新增劳动力（每年自然增长）
- 院校培养速度
- 培训补贴影响

**企业需求侧**：
- 企业数量（受投资环境影响）
- 行业景气指数（周期性波动）
- 薪资期望（随时代升高）

**匹配算法**：
```
就业成功率 = f(学历匹配度, 技能匹配度, 薪资合理性, 运气值)
```

### 3.3 详细数值模型

**企业招聘满意度公式**：
```
企业招聘满意度 = 0.3×学历匹配度 + 0.4×技能匹配度 + 0.2×薪资合理性 + 0.1×经验匹配度
```

**求职者就业概率公式**：
```
就业概率 = 基础匹配度 × 运气系数 × (1 + 就业促进buff) × 行业景气系数
```

**科技进步公式**：
```
科技进步 = Σ(研发投入 × 研发效率系数) + 随机事件修正 + 人才溢出效应
```

**GDP增长公式**：
```
GDP增长 = Σ(就业人口 × 人均产出) × (1 + 科技进步加成) - 失业人口消耗 - 通胀损耗
```

## 四、人才系统

### 4.1 求职者属性

每个求职者拥有四个维度属性（1-100）：

| 属性 | 说明 | 对就业影响 |
|------|------|------------|
| 学历 | 教育程度 | 影响高薪岗位解锁 |
| 技能 | 专业技能水平 | 影响岗位匹配度 |
| 经验 | 工作年限 | 影响薪资期望 |
| 运气 | 随机因素 | 不可控波动 |

### 4.2 求职者生命周期
1. **出生** → 进入待就业池
2. **求学阶段** → 可选择投入教育资源
3. **求职阶段** → AI自动匹配或受政策buff影响
4. **就业/失业** → 就业者产出GDP，失业者消耗社会资源
5. **退休/死亡** → 退出劳动力市场

### 4.3 院校与培训
- **小学/中学**：提升基础学历
- **职业学院**：提升特定技能
- **大学**：高学历 + 宽泛技能
- **研究所**：科研人才产出

### 4.4 人才流动机制
- **人才引进**：玩家可花费资源从"国际市场"引进人才（加速但昂贵）
- **人才流失**：政策不当会导致本地人才流向其他地区
- **海归潮**：特定事件触发海外人才回归

## 五、企业系统

### 5.1 行业细分

游戏包含6大行业，每个行业有独立的景气指数（周期性波动）：

| 行业 | 时代 | 核心岗位 | 薪资水平 |
|------|------|----------|----------|
| 农业 | 原始时代 | 农民、畜牧 | 低 |
| 制造业 | 蒸汽+电气 | 工人、技术员 | 中 |
| 商业 | 手工场+电气 | 商人、销售 | 中 |
| 服务业 | 电气+信息 | 客服、金融 | 中高 |
| 科技业 | 信息+智能 | 工程师、研究员 | 高 |
| 公共事业 | 全时代 | 公务员、教师 | 稳定中等 |

### 5.2 企业类型（按时代解锁）

| 时代 | 企业类型 | 岗位需求特征 |
|------|----------|--------------|
| 原始劳动 | 农场、作坊 | 体力型，学历要求低 |
| 手工场 | 工场、商铺 | 技艺型，学徒制 |
| 蒸汽革命 | 工厂、矿山 | 技能型，需职业培训 |
| 电气时代 | 跨国公司 | 管理+技术双需求 |
| 信息时代 | 科技公司 | 高学历，技能专精 |
| 大智能时代 | AI实验室 | 顶尖人才，创新驱动 |

### 5.3 企业行为
- **招聘**：根据岗位需求发布招聘信息
- **涨薪/降薪**：根据经营状况调整
- **扩张/收缩**：受经济环境影响
- **倒闭**：连续亏损则破产，释放失业人口

### 5.4 企业竞争与生命周期
- **企业竞争**：同行业企业会竞争人才，薪资高的企业更容易招到人
- **企业成长**：企业每年有成长机会（规模扩大、转型）
- **行业周期**：每个行业有繁荣→衰退→复苏的周期
- **倒闭连锁**：大企业倒闭会引发上下游连锁反应

## 六、玩家干预系统

### 6.1 政策工具

**财政政策**：
- 税率调整（影响企业盈利与个人收入）
- 财政补贴（特定行业/群体）
- 政府支出（基建、教育投入）

**教育政策**：
- 教育经费投入
- 学制长短调整
- 专业导向（理工/文科比例）

**就业政策**：
- 最低工资标准
- 失业保险制度
- 就业促进计划（局部buff）

**科技政策**：
- 研发补贴
- 技术引进
- 知识产权保护

### 6.2 基础建设

玩家可在地图上建造：

| 建筑 | 效果 |
|------|------|
| 学校 | 增加人才培养速度 |
| 培训中心 | 提升技能匹配度 |
| 企业园区 | 吸引企业入驻 |
| 研发中心 | 加速科技进步 |
| 就业中心 | 提升就业匹配率 |

### 6.3 干预限制与政策延迟

| 政策类型 | 延迟时间 | 副作用 |
|----------|----------|--------|
| 税率调整 | 6个月 | 影响消费信心 |
| 教育投入 | 3-5年 | 人才供给滞后 |
| 最低工资 | 1年 | 可能推高失业率 |
| 研发补贴 | 即时+长期 | 财政负担累积 |
| 基建投资 | 2-3年 | 短期无效果 |

### 6.4 区域地理系统
- **地图分区**：将社会地图划分为4-6个区域（如：东区、西区、南区、北区、中心区）
- **区域差异**：不同区域有初始禀赋（资源、人口密度、交通便利度）
- **区域发展不平衡**：玩家可通过基建投资平衡区域发展
- **区域特色企业**：特定区域解锁特定行业（如港口区解锁商业）

## 七、事件系统

### 7.1 随机事件类型

| 类型 | 事件示例 | 影响 |
|------|----------|------|
| 经济危机 | 1929大萧条 | 企业倒闭潮，失业率飙升 |
| 技术突破 | 互联网发明 | 新行业崛起，人才需求剧变 |
| 人口事件 | 婴儿潮/老龄化 | 劳动力供给大幅变化 |
| 社会事件 | 罢工运动 | 生产停滞，经济受损 |
| 政策变革 | 改革开放 | 市场经济激活 |

### 7.2 事件触发机制
- 随时间推移自动触发
- 特定条件满足时触发（如失业率>30%触发福利改革运动）
- 玩家政策选择触发（如提高失业率触发社会动荡）

## 十一、新手引导与难度系统

### 11.1 难度等级

| 难度 | 描述 | 特征 |
|------|------|------|
| 和谐模式 | 新手友好 | 事件温和、预算宽松、失业率容忍度高 |
| 普通模式 | 标准体验 | 正常事件、预算平衡 |
| 挑战模式 | 硬核模拟 | 随机危机频繁、预算紧张、玩家干预延迟更长 |

### 11.2 新手引导流程

```
引导流程：
1. 第一年：只显示地图，提示"这是你的社会"
2. 第二年：弹出"需要建设学校培养人才"
3. 第三年：弹出"企业需要工人，点击查看"
4. 第四年：弹出"你可以调整税率来影响经济"
5. 第五年：引导结束，正式进入自由模式
```

## 十二、成就与可玩性系统

### 12.1 成就系统

| 成就 | 条件 | 奖励 |
|------|------|------|
| 就业率达到100% | 任意年份失业率为0 | 称号"就业之父" |
| 科技进步100% | 进入大智能时代 | 称号"文明推进者" |
| 基尼系数<0.2 | 持续5年 | 称号"公平先锋" |
| 百年不衰退 | 连续100年GDP正增长 | 称号"常青经济" |
| 全行业繁荣 | 6个行业同时景气 | 称号"六脉神剑" |

### 12.2 存档与设置
- **多存档槽位**：支持3个独立存档
- **游戏设置**：音乐音量、音效音量、文字大小、加速倍数
- **游戏速度**：0.5x / 1x / 2x / 4x / 暂停
- **回退功能**：付费存档（消耗资源）可回退1年
